# Student Management System (Python OOP Project)

## 📌 Project Overview

The Student Management System is a Python-based project built using Object-Oriented Programming (OOP) principles.
It allows users to manage student records efficiently through a command-line interface. The system provides functionality to add, view, search, update, and delete student records.

This project demonstrates the use of Python classes, file handling, and modular programming to build a simple but practical management system.

---

## 🚀 Features

* Add new student records
* Display all students
* Search for a student by ID or name
* Update student details
* Delete student records
* Data storage using JSON file
* Menu-driven command line interface
* Structured using OOP concepts

---

## 🛠 Technologies Used

* Python
* Object-Oriented Programming (OOP)
* JSON file handling
